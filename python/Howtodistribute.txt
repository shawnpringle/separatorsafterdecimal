Python Module Distribution
--------------------------

Go into the doc directory and type 'make text'.

    Install pip and then:
    
    python -m pip install --user --upgrade setuptools wheel sphinx
    python -m pip install --user --upgrade twine

Then run:
    
    python setup.py sdist bdist_wheel
    python -m twine upload -u Shawn.Pringle --repository-url https://test.pypi.org/legacy/ dist/*
    
Supply the password that you use on the test site.

Go to some temporary directory like C:\TEMP and type: 

    python -m virtualenv ENV
    
Run:
    cd ENV
    cmd
    ENV\Scripts\activate.bat
    
    Then do: python -m pip install --index-url https://test.pypi.org/simple/  qsdn
    
    Now try:
    python
      >>> import qsdn
      >>> le = qsdn.QSDNLocale('en_US')
      >>> le.toDecimal("14,050")
      (Decimal('14050'), True)
      >>> le.toDecimal("0.000,001", 10)
      (Decimal('0.000001'), True)
      >>> ans = le.toDecimal("0.000,000,1",10)
      >>> ans
      (Decimal('1E-7'), True)
      >>> le.toString(ans[0])
      '0.000,000,1'
      >>> quit()
      
   copy test.py from the source distribution to this ENV directory and run
   python -m unittest
   
   
   All tests should finish OK.
   
   
If there have been no failures with the tests, it is time to make a real release:
    get back to the source distribution directory in python
    python -m twine upload -u Shawn.Pringle --repository-url https://test.pypi.org/legacy/ dist/*
